# Example 05: Issue Tracking

This example demonstrates how to use `@it-supervisor/issue-manager` for issue management.

## What This Example Does

1. Creates and manages issues
2. Searches and filters issues by various criteria
3. Adds comments and attachments
4. Generates statistics
5. Exports issues to CSV

## Running the Example

```bash
npm install
npm run build
npm start
```

See the [main examples README](../README.md) for detailed instructions.

## Key Features

- ✅ Full CRUD operations
- ✅ Advanced search and filtering
- ✅ Comment system with attachments
- ✅ Priority calculation
- ✅ Statistics and aggregation
- ✅ CSV export
- ✅ Label management
