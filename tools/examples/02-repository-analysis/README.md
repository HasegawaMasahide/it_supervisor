# Example 02: Repository Analysis

This example demonstrates how to use `@it-supervisor/repo-analyzer` to analyze Git repositories.

## What This Example Does

1. Analyzes a local Git repository for:
   - Programming languages and their distribution
   - Code complexity metrics
   - Dependency graphs
   - Framework detection
   - Entry point detection
2. Calculates repository statistics
3. Generates a comprehensive analysis report

## Running the Example

```bash
npm install
npm run build
npm start
```

See the [main examples README](../README.md) for detailed instructions.

## Key Features

- ✅ Language detection and statistics
- ✅ Complexity calculation (cyclomatic complexity)
- ✅ Framework and dependency detection
- ✅ Entry point analysis
- ✅ Metadata extraction (README, LICENSE, etc.)
