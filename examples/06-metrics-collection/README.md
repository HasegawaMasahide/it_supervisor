# Example 06: Metrics Collection

This example demonstrates how to use `@it-supervisor/metrics-model` for metrics tracking.

## What This Example Does

1. Records metrics (numeric, string, boolean)
2. Queries metrics with filters
3. Aggregates metrics by category
4. Compares metrics over time
5. Exports/imports metrics

## Running the Example

```bash
npm install
npm run build
npm start
```

See the [main examples README](../README.md) for detailed instructions.

## Key Features

- ✅ Multi-type metrics (numeric, string, boolean)
- ✅ Time-series data storage
- ✅ Advanced querying and filtering
- ✅ Aggregation functions (count, min, max, avg, sum)
- ✅ Before/after comparisons
- ✅ JSON and CSV export/import
- ✅ Transaction support
