# Example 03: Report Generation

This example demonstrates how to use `@it-supervisor/report-generator` to create professional reports.

## What This Example Does

1. Generates reports in multiple formats:
   - HTML with embedded charts
   - Markdown
   - PDF (using Puppeteer)
2. Uses custom templates
3. Supports multi-language reports

## Running the Example

```bash
npm install
npm run build
npm start
```

See the [main examples README](../README.md) for detailed instructions.

## Key Features

- ✅ HTML, Markdown, and PDF export
- ✅ Chart.js integration for data visualization
- ✅ Custom template support
- ✅ Multi-language support
- ✅ Table of contents generation
