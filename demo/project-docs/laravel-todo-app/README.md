# Laravel TODO App (監査デモ用)

このアプリケーションは、IT資産監査・改善サービスのデモンストレーション用に作成されたサンプルアプリです。

## ⚠️ 重要な注意事項

**このアプリには意図的に以下の問題が含まれています：**

### セキュリティ問題
- SQLインジェクションの脆弱性（生のクエリ使用）
- XSS（クロスサイトスクリプティング）脆弱性
- CSRF対策の不備
- 機密情報のハードコーディング
- 不適切な認証実装

### コード品質問題
- 循環的複雑度が高い関数
- 重複コード
- 長すぎる関数（100行超）
- マジックナンバーの使用
- 適切でないエラーハンドリング

### パフォーマンス問題
- N+1クエリ問題
- 不要なデータの取得
- インデックスの欠如

### 依存関係問題
- 古いLaravelバージョン（8.x）
- セキュリティ脆弱性のあるパッケージ

## 機能

- ユーザー登録・ログイン
- TODOタスクの作成・編集・削除
- タスクの完了/未完了切り替え
- タスクの優先度設定
- タスクの検索・フィルタリング

## セットアップ

```bash
# 依存関係のインストール
composer install

# 環境設定
cp .env.example .env
php artisan key:generate

# データベースマイグレーション
php artisan migrate

# 開発サーバー起動
php artisan serve
```

## 監査の実施

このアプリケーションを監査するには、プロジェクトルートの `/tools` ディレクトリにあるツール群を使用してください。

詳細な手順は `/demo/ANALYSIS_GUIDE.md` を参照してください。
